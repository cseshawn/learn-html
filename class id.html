Difference Between ID and Class

  • ID is unique - can be used only once per page
  • Class can be used multiple times on same page
  • ID is used for specific, one-of-a-kind elements
  • Class is used for grouping similar elements
  • ID can be used in URLs for linking (anchor)
  • Class is commonly used for styling multiple elements

Summary

Feature ID Class
Uniqueness Unique (one per page) Can be used many times
Syntax id="name" class="name"
CSS selector #name .name
Multiple values No (only one ID per element) Yes (space-separated classes)
Anchor linking Yes (href="#id") No
