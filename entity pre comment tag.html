HTML Entities, Pre Tag, and Comments

HTML Entities

Less than: < (uses &lt;)

Greater than: > (uses &gt;)

Ampersand: & (uses &amp;)

Double quote: " (uses &quot;)

Single quote: ' (uses &apos;)

Copyright: © (uses &copy;)

Registered: ® (uses &reg;)

Euro: € (uses &euro;)

Pound: £ (uses &pound;)

Yen: ¥ (uses &yen;)

Degree: ° (uses &deg;)

Plus/minus: ± (uses &plusmn;)

Multiplication: × (uses &times;)

Division: ÷ (uses &divide;)

Space:   (non-breaking space using &nbsp;)

Pre Tag (Preformatted Text)

        This is preformatted text.
        It preserves    spaces   and   tabs.
        Line breaks are also preserved.

        Exactly as written in the code.
        Like a poem or code snippet.

Pre tag for code:

        function hello() {
            console.log("Hello World!");
            return true;
        }

Pre tag showing HTML code:

        <html>
            <head>
                <title>My Page</title>
            </head>
            <body>
                <h1>Hello World</h1>
            </body>
        </html>
